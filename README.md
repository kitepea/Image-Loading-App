# Simple Image Loading App

**Tech Stack**
* Clean Architecture with MVVM and DI (Hilt)
* UI & Animation: Modern Jetpack Compose
* Remote API: Retrofit, OkHttp
* 3rd-party Image Loader: Coil
* Other: Pagination for Scroller
